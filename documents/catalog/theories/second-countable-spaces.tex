\section{第二可算空間}
\label{sec:second-countable-spaces}

\begin{definition}
	位相空間$ X $において可算な開集合基が存在するとき
	$ X $は第二可算条件を満たすといい, このとき$ X $が
	indexjj{だいにかさん}{第二可算}{second countable}であるという.
\end{definition}
